# 作業ルール

このプロジェクトを修正する場合は、作業前に以下のObsidianノートを読む。

- `/Users/nao/Documents/Obsidian Vault/dog-image-generator/dog-image-generator.md`

作業後は必ず以下を行う。

1. 実装内容に合わせて上記Obsidianノートを更新する。
2. `npm run build`、`npm run lint`、`npx tsc --noEmit -p api/tsconfig.json`を実行する。
3. 変更をコミットする。
4. `origin`が`https://github.com/naokoni24/dog_event_image.git`であることを確認する。
5. 現在の`master`ブランチを`origin/master`へpushする。確認は不要。
